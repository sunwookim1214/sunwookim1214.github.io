# sunwookim1214.github.io

Personal academic website of Sunwoo Kim, available at [sunwookim1214.github.io](https://sunwookim1214.github.io).

Built with the [al-folio](https://github.com/alshedivat/al-folio) Jekyll theme. See the al-folio repository for [customization docs](https://github.com/alshedivat/al-folio/blob/main/CUSTOMIZE.md) and [FAQ](https://github.com/alshedivat/al-folio/blob/main/FAQ.md).

## Editing

- Homepage: `_pages/about.md`
- Site settings: `_config.yml`
- Social links: `_data/socials.yml`
- Publications: `_bibliography/papers.bib`
- News items: `_news/`

Pushing to `main` triggers the deploy workflow, which builds the site and publishes it to the `gh-pages` branch.
